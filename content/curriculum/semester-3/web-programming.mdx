### 🎯 أهداف المقرر

بنهاية هذا المقرر، سيكون الطالب قادراً على:

- **بناء** صفحات ويب هيكلية باستخدام HTML الدلالية.
- **تنسيق** وتصميم واجهات المستخدم باستخدام CSS.
- **إضافة** عناصر تفاعلية باستخدام JavaScript.
- **تطبيق** مفاهيم التصميم المتجاوب لمختلف الأجهزة.
